4.4 Exportation

Pour les exporter avec le paquet writexl :

iris %>% tbl_summary(by=Species) %>%
  gtsummary::as_tibble() %>% 
  writexl::write_xlsx(., "example_gtsummary1.xlsx")

Sinon

iris %>% tbl_summary(by=Species) %>%
  gtsummary::as_gt() %>% 
  gt::gtsave(., "example_gtsummary3.rtf")
